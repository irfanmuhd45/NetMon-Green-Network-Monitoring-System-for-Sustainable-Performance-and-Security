# Green Network Monitoring System for Sustainable Performance and Security

A full-stack network monitoring platform designed for real-time monitoring of network devices using SNMP telemetry, live alerts, and analytics dashboards.

## Features

- Real-time device monitoring
- SNMP polling
- CPU and memory monitoring
- Interface utilization monitoring
- Telegram alert notifications
- Live dashboard updates using Socket.IO
- Cisco device support
- GNS3 lab integration

## Technologies

- Node.js
- Express.js
- React
- Socket.IO
- SNMP
- SQLite
- GNS3
- Cisco IOS
- Telegram Bot API
